# CardSchemeDefinitionsSDK

## Overview

## list

Fetch a list of the definitions of each card scheme.

### Example Usage

```python
from gr4vy import Gr4vy
import os


with Gr4vy(
    bearer_auth=os.getenv("GR4VY_BEARER_AUTH", ""),
) as g_client:

    res = g_client.card_scheme_definitions.list()

    # Handle response
    print(res)

```
